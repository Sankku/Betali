## 🎯 Por Dónde Empezar

Si eres nuevo en el proyecto:
1. Lee `architecture/SAAS_ARCHITECTURE.md` para entender la arquitectura
2. Revisa `architecture/BETALI_MCP_DOCS.md` para patrones y best practices
3. Consulta `roadmaps/MVP-STATUS-REPORT.md` para estado actual
4. Revisa los PRDs relevantes en `/prds/` según el feature que trabajes
